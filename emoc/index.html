Emocracies:

Advanced Technologies for Democratic Organizations

What is an "emocracy"?

To a first order, an emocracy is exactly what the term was coined to suggest: an "electronic democracy."  More specifically, an emocracy is an organization built around the general democratic goals of fairness and bottom-up control by the membership, but which takes maximum advantage of the power of modern technology in order to achieve these goals more effectively.  An emocracy does not just use technology to do conventional things more efficiently - such as by using E-mail or streaming audio/video as means of communication between members or by using computers to count votes - but uses technology to enable fundamental new and more effective methods of organization.

The Emocracies Project

This web page is intended to be the seed of a project to explore all the ways in which technology is being or could be used to make democratic organizations function better, and to brainstorm and develop new technologies (e.g., open-source software) specifically for this purpose.  Initially, it is a personal research project by Bryan Ford, the author of this page; however, I hope that over time other people will become interested and that the project will gradually evolve into a collective effort and become an emocracy in its own right.
